# What is ChannelFinder
ChannelFinder is a directory service that allows you to search for channels by name, tags, and properties. It enables them to be represented in a hierarchical view for applications (by function or location), as the EPICS channel protocol has a flat structure by default.

ChannelFinder is designed for use with other EPICS services, such as Phoebus, OpenAPI and PVinfo. 

It also has a web interface for manual operation and debugging.
This interface allows tags or properties to be managed manually for archiving or correcting errors.
